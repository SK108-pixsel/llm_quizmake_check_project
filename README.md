# AI Quiz Generator
ローカルLLM（Ollama）とメタ検索エンジン（SearXNG）を活用し、競技クイズ用の問題文を完全自動で生成・評価するシステムです。
## 概要
あらかじめ用意した単語リスト（`labeled_entities.txt`）からランダムに単語を抽出し、指定されたジャンルに合致するかをAIが判定します。
ジャンルに合致した単語についてSearXNGでWeb検索を行い、得られた背景情報を元にLLMがクイズ問題を作成します。
さらに、作成された問題文に対して別のプロンプトで「正確性・限定性」「文章の構造」「難易度と面白さ」の3観点から200点満点で評価を行い、合格基準（160点）に達した問題をCSVに出力します。
## 特徴
- **完全ローカル動作**: Ollamaを使用することで、API利用料をかけずにローカル環境でテキスト生成と評価を完結させます。
- **自動ジャンル判定**: 抽出した単語が選択したカテゴリ（歴史、科学、アニメなど9種類）に属するかを自動でスクリーニングします。
- **検索連動型（RAG）問題作成**: SearXNGを用いた検索結果をプロンプトに組み込むことで、事実に基づいた情報密度の高い問題を作成します。
- **厳格な自動評価・修正ループ**: 作成された問題が基準を満たさない場合、AIによるフィードバック（理由）を元に自動で再生成を試みます。
## 動作要件
- Python 3.x
- [Ollama](https://ollama.com/)（ローカル環境で起動していること）
- 推奨LLMモデル: `qwen3.5:4b`（OllamaにPull済みであること）
- 必要なPythonパッケージ: `ollama`
## インストールと準備
1. 本リポジトリをローカルにクローンします。
2. 依存パッケージをインストールします。（例: `pip install ollama`）
3. Ollamaを起動し、設定ファイル（`quiz_config.py`）で指定されているモデルをダウンロードします。（例: `ollama pull qwen3.5:4b`）
4. クイズの正解となる単語リストを記述した `labeled_entities.txt` をプロジェクトのルートディレクトリに配置します（1行につき1単語）。
## 使用方法
メインスクリプトである `generate_quiz.py` を実行します。
```bash
python generate_quiz.py
```
ターミナル上で作成したいクイズのジャンル（1〜9）を選択すると、自動的に問題作成と評価のループが開始されます。
合格した問題データは、同ディレクトリの `quiz_results_evaluated.csv` に追記保存されます。
## ファイル構成
- `generate_quiz.py`: メインの実行スクリプト。ユーザー入力からジャンル判定、検索、生成、評価、CSV保存までの全体フローを制御します。
- `quiz_config.py`: 定数定義、システムプロンプト（生成用・評価用）、単語リストの読み込み機能を管理する設定ファイルです。
- `quiz_score.py`: SearXNGでの検索実行、Ollama APIを介したLLMとの通信、JSONデータの抽出、問題生成および評価のコアロジックを担当します。
- `genre_judge.py`: 抽出された単語が、指定されたジャンルに属するかどうかをLLMを用いて判定するモジュールです。
- `test_connection.py`: ローカルのOllama API（`http://localhost:11434`）への接続およびモデル（`qwen2.5-coder:latest`）の動作確認を行うテストスクリプトです。
